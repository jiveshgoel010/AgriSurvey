# 🌾 AgriSurvey App

AgriSurvey is an Android application designed to help collect, analyze, and visualize agricultural data from farmers across different Indian states. The goal is to optimize land usage and improve farming practices through data-driven insights.

---

## 🚀 Key Features

- 📋 **Survey Collection**: Farmers can submit data about landholding size, crop type, irrigation method, and more.
- 📍 **Location-Aware Surveys**: Option to capture city/state automatically.
- 📊 **Insights & Visualization**: Visual charts and heatmaps based on state-wise survey data.
- 🌐 **State-Wise Navigation**: View agricultural trends per Indian state.
- 🤖 **AI-Powered Queries (Gemini)**: Ask farming and weather-related questions using integrated AI.
- 🗺️ **Google Maps Integration**: View geospatial agricultural data on maps (coming soon).
- 🔐 **Authentication**: Sign in with Google or OTP-based phone login.
- 👤 **User Profiles**: Save and manage user profiles with name, email, and profile photo.
- 🧠 **MVVM Architecture**: Clean architecture with separation of concerns.
- 🧭 **Bottom Navigation**: Smooth navigation between Home, Survey, Insights, and Profile screens.
- 🎨 **Material Design 3 UI**: Colorful, modern, and responsive interface.

---

## 🧰 Tech Stack

### 🛠️ Development
- **Language**: Kotlin
- **Architecture**: MVVM (Model-View-ViewModel)
- **UI Design**: XML + Material 3 Components
- **Navigation**: Jetpack Navigation + Fragments

### ☁️ Backend
- **Firebase Authentication** (Google & OTP)
- **Firebase Realtime Database** (Survey Data)
- **Firebase Firestore** (User Profiles)

### 📍 APIs & Libraries
- **Google Maps API** – for geospatial data display
- **Gemini API** – for AI Q&A capabilities
- **MPAndroidChart** – for data visualization
- **View Binding** – for safer and cleaner UI code
- **Glide** – for image loading
- **Material Components** – for polished UI/UX
